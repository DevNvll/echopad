# Echopad

A local-first, chat-style note-taking app for quick thoughts and ideas.

</div>

### Features

- **Vault-based organization** — Your notes live in local folders as plain markdown files. Multiple vaults supported.
- **Nested notebooks** — Organize notes hierarchically with notebooks and sub-notebooks
- **Quick Capture** — Global shortcut (`Alt+Space`) to easy capture notes anywhere
- **Tags** — Use `#hashtags` inline; search and filter by tag
- **Link previews** — URLs are automatically enriched with OpenGraph metadata
- **Image support** — Paste images directly into notes
- **Favorites & pinned notebooks** — Quick access to what matters
- **Command palette** — Fast navigation and actions with keyboard shortcuts
- **Per-vault customization** — Custom accent colors and icons for each vault
- **Auto-updates** — Built-in updater keeps the app current
